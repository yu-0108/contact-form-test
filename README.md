・アプリケーション概要</br>
購入した商品などに対して問い合わせしたい時に使用できる問い合わせフォーム

・機能一覧</br>
①問い合わせ内容の確認画面
入力画面で入力した内容の確認画面。
修正したい場合は、修正ボタンで全画面に戻ることができます。
<img width="1440" alt="スクリーンショット 2023-08-27 0 27 38" src="https://github.com/yu-0108/contact-form-test/assets/135131310/c7bb9ec0-c048-4f64-80b8-0d0f42aa70dd">

②送信完了画面
入力した内容を問題なく送信できたら送信完了の画面を表示します。
トップページに戻るボタンも実装。
